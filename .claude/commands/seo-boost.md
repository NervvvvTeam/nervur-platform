# SEO Boost Agent — NERVÜR Platform

Maximize SEO performance across the entire site.

## Technical SEO:
1. **Meta tags** — Verify every page has unique title, description, canonical, OG tags
2. **Sitemap** — Verify public/sitemap.xml is up to date with all pages
3. **Robots.txt** — Verify public/robots.txt allows indexing of public pages, blocks /app/
4. **JSON-LD** — Verify structured data on homepage (Organization, WebSite, SoftwareApplication for each tool)
5. **Page speed** — Check bundle sizes, suggest optimizations
6. **Internal linking** — Blog articles should cross-link each other and link to tool pages
7. **Heading hierarchy** — Every page should have exactly one H1, proper H2/H3 structure

## Content SEO:
1. **Blog articles** — Verify 3 articles exist with proper SEO (e-reputation, cybersecurite, performance-web)
2. **Keywords** — Each page should target specific keywords
3. **Alt tags** — All images should have descriptive alt text
4. **URL structure** — Clean, descriptive URLs

## Google Search Console:
1. **Sitemap submitted** — Verify at https://search.google.com/search-console
2. **No crawl errors** — Check for 404s or blocked pages
3. **Mobile usability** — No mobile issues

## Fix every issue found. Generate a final SEO score (0-100).
